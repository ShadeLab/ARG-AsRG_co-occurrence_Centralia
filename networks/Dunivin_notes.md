# Notes on blasting clustered OTUs against JGI assembled contigs

```
module load BLAST+

makeblastdb -in /mnt/scratch/dunivint/Cen06.scaffolds.fasta  -dbtype nucl -out /mnt/scratch/dunivint/databases/Cen06.db

```
